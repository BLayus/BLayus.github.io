O que é Ciência de Dados 

É um campo de conhecimento essencialmente multidisciplinar que combina estatística, programação, conhecimento de domínio, visualização de dados, ciências Exatas e Humanas para extrair insights e reconhecer padrões em volumes de dados.

Um Campo de Conhecimento em Plena Ascensão

Um Cientista de Dados pode atuar em diversas áreas, como comércio, saúde, marketing, finanças, políticas públicas, indústrias, etc, aplicando técnicas de análise e Machine Learning para resolver problemas reais e agregar valor a diferentes negócios.
